Comment: `ctrl + K and ctrl + C`
Decomment: `ctrl + K and ctrl + U`

Indent: `ctrl + ]`
Dedent: `ctrl + [`

Fold: `ctrl + shift + [` , `ctrl + K and ctrl + 0`
Unfold:`ctrl + shift + ]` , `ctrl + K and ctrl + J`

<br>

- Box Selection
  - `shift + Arrow`
  - `shift + alt + click`
  - `move middle mouse`
- Add a cursor
  - `ctrl + alt + Arrow`
  - `alt + click`
- Create cursor on all occurrence of a string
  - `ctrl + shift + L`
<br>

Move a line: `alt + Arrow`
Copy a line: `shift + alt + Arrow`
Delete a line: `ctrl + shift + K`

Rename Refactor: `F2`
Formatting: `shift + alt + F`, `ctrl + K and ctrl + F`

Split editor: `ctrl + \`
Terminal: ctrl + `
