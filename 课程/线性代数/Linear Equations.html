Linear Equations

зәҝжҖ§ж–№зЁӢз»„¶

йҪҗж¬ЎзәҝжҖ§ж–№зЁӢз»„¶

е®ҡзҗҶпјҡи§Јз©әй—ҙзҡ„з»ҙж•° и®ҫ \(A\in M_{m\times n}(F)\)пјҢ\(\text{r}(A) = r\)пјҢ еҲҷйҪҗж¬ЎзәҝжҖ§ж–№зЁӢз»„зҡ„и§Јз©әй—ҙ \(N(A)\) зҡ„з»ҙж•° \(\dim N(A) = \dim(\text{null}A) = n-r\)

и®°и§Јз©әй—ҙзҡ„еҹә \(X_1,X_2,\cdots,X_{n-r}\) дёәзәҝжҖ§ж–№зЁӢз»„ \(AX=b\) зҡ„еҹәзЎҖи§Јзі»

и®° \(R(A)\) дёә \(n\) дёӘеҲ—еҗ‘йҮҸеј жҲҗзҡ„еҲ—з©әй—ҙпјҢ\(R(A^T)\) дёә \(m\) дёӘиЎҢеҗ‘йҮҸеј жҲҗзҡ„иЎҢз©әй—ҙпјҢ еҲҷз”ұ \(\dim R(A^T) + \dim N(A) = n\) еҸҜеҫ—пјҢ\(N(A) = (R(A^T))^{\perp}\)

дҫӢйўҳ и®ҫ \(A\) жҳҜ \(m\times n\) е®һзҹ©йҳөпјҢиҜҒжҳҺпјҡ\(r(A^TA)=r(A)\)

\(\text{Proof :}\) жҳҫз„¶ $\text{r}(A^TA) \le \text{r}(A) $пјҢж•…еҸӘйңҖиҜҒ $\text{r}(A^TA) \ge \text{r}(A) $пјҢеҚіиҜҒ \(N(A^TA) \subseteq N(A)\)пјҢ $(A^TA)\bm{X} = 0 \ \Rightarrow \bm{X}T(ATA)\bm{X} = 0 \ \Rightarrow (\bm{X}TAT)(A\bm{X}) = 0 \ \Rightarrow |A\bm{X}|^2 = 0 \ \Rightarrow A\bm{X} = 0 $

йқһйҪҗж¬ЎзәҝжҖ§ж–№зЁӢз»„¶

йқһйҪҗж¬ЎзәҝжҖ§ж–№зЁӢз»„ \(A\bm{X} = \bm{b}\) зҡ„дёҖиҲ¬и§Јдёә \(\bm{X} = \bm{X_0 + \overline X}\)пјҢ е…¶дёӯ \(X_0\) дёә \(A\bm{X} = \bm{b}\) зҡ„дёҖдёӘзү№и§ЈпјҢ\(\overline{X}\) дёә \(A\bm{X} = 0\) зҡ„дёҖиҲ¬и§ЈгҖӮ

еҜ№дәҺйқһйҪҗж¬ЎзәҝжҖ§ж–№зЁӢз»„пјҢдёӢеҲ—е‘Ҫйўҳзӯүд»·пјҡ \(\pod 1\; A\bm{X} = \bm{b}\)жңүи§Ј \(\pod 2\; b\in R(A)\) \(\pod 3\; \text{r}(A, \bm{b}) = \text{r}(A)\)
